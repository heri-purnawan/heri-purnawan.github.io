## Deskripsi Mata Kuliah / *Course Description*

Mata kuliah ini bertujuan memberikan dasar-dasar pemrograman komputer, mencakup konsep algoritma, logika pemrograman, dan implementasinya menggunakan bahasa pemrograman. Topik meliputi pengantar pemrograman, algoritma, variabel, tipe data, operasi aritmatika, logika percabangan, perulangan, fungsi, dan array. Mahasiswa akan mempelajari dan mengimplementasikan konsep-konsep tersebut untuk menyelesaikan permasalahan sederhana dalam berbagai domain.

*This course introduces the fundamentals of computer programming, focusing on algorithm concepts, programming logic, and implementation using programming languages. Key topics include programming basics, algorithms and flowcharts, variables, data types, arithmetic operations, branching logic, loops, functions, and arrays. Students will apply these concepts to solve simple problems in their domains, especially electrical engineering.*
